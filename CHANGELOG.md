# Change Log

All notable changes to the "vscode-cocktail" extension will be documented in this file.

## [1.1.0]

- Added fixed background and foreground imagery with x-axis repeat support
- Added bartender animation layer using walk/wait GIFs and randomized walk/wait modes
- Improved bartender movement logic to avoid edge collisions and switch states every 1.5 seconds
- Added confetti effect when selecting random drinks
- Adjusted drink display to fix the image to the bottom of its panel
- Moved recipe panel to the upper-right corner for clearer layout

## [1.0.0]

- Display cocktail listings
- View cocktail ingredients, flavor notes, and preparation methods
- Randomly select a cocktail
- Show the current cocktail's flavor or preparation instructions
- Supports localization with built-in `en-US`, `ja-JP`, and `zh-TW`
